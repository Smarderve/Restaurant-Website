# Restaurant-Website

A simple website for a restaurant featuring menus, contact information, and responsive design. This project includes HTML, CSS, and JavaScript files along with assets such as images and fonts.

## 📁 Structure

- `index.html` – the main landing page.
- `assets/` – contains fonts and images used by the site.
- `src/scripts/script.js` – JavaScript logic for interactive features.
- `src/styles/style.css` – custom styles for layout and appearance.

## ⚙️ Setup

No build tools are required. Simply open `index.html` in a web browser to view the site. If you plan to develop further:

1. Clone the repository to your local machine.
2. Edit HTML, CSS, or JS files as needed.
3. Refresh the browser to see changes.

## 🚀 Features

- Responsive layout compatible with desktop and mobile.
- Styled components using external CSS.
- Basic JavaScript interactivity.

## 🛠️ Development

- Modify `src/styles/style.css` for styling changes.
- Add scripts or update logic in `src/scripts/script.js`.
- Place new images or fonts into the `assets/` directory and reference them in code.
